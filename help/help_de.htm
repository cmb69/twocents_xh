Voraussetzungen

Twocents_XH ist ein Plugin für CMSimple_XH. Es benötigt CMSimple_XH ≥ 1.6 und PHP ≥ 5.1.

zum Seitenanfang

Installation

Die Installation erfolgt wie bei vielen anderen CMSimple_XH-Plugins auch. Im CMSimple_XH Wiki finden sie ausführliche Hinweise.

  1. Sichern Sie die Daten auf Ihrem Server.
  2. Entpacken Sie die ZIP-Datei auf Ihrem Computer.
  3. Laden Sie das gesamte Verzeichnis twocents/ auf Ihren Server in das plugins/ Verzeichnis von CMSimple_XH hoch.
  4. Vergeben Sie Schreibrechte für die Unterverzeichnisse css/ und languages/.
zum Seitenanfang

Einstellungen

Die Konfiguration des Plugins erfolgt wie bei vielen anderen CMSimple_XH-Plugins auch im Administrationsbereich der Homepage. Wählen Sie Plugins → Twocents.

Die Lokalisierung wird unter "Sprache" vorgenommen. Sie können die Zeichenketten in Ihre eigene Sprache übersetzen (falls keine entsprechende Sprachdatei zur Verfügung steht), oder sie entsprechend Ihren Anforderungen anpassen.

Das Aussehen von Twocents_XH kann unter "Stylesheet" angepasst werden.
